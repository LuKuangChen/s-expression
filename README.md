# S-expression

An S-expression parser written in ReScript.

## Development

`npm run dev` and then start editing the code.

## Build

```sh
npm run build
```

## Supported Features

- Strings
- Symbols
- Lists in `( )`
- Lists in `[ ]`
- Vectors
- line comments `;`
- S-expression comments `#;`

## Not supported feature

- range comments `#| |#`

## Installation

```sh
npm install git+https://github.com/brownplt/s-expression.git
```
