# S-expression

An S-expression parser written in ReScript.

## Supported Features

- Strings
- Symbols
- Lists in `( )`
- Lists in `[ ]`
- Vectors

## Not supported feature

- all sorts of comments:
  - line comments `; `
  - block comments `#| |#`
  - S-expression comments `#; `

## Installation

```
npm i @lukuangchen/s-expression
```
