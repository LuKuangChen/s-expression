# S-expression

An S-expression parser written in ReScript.

## Development

`npm run dev` and then start editing the code.

## Build

`npm run build`

## Supported Features

- Strings
- Symbols
- Lists in `( )`
- Lists in `[ ]`
- Vectors
- line comments `;`
- S-expression comments `#;`

## Not supported feature

- range comments `#| |#`

## Installation

```
npm i @lukuangchen/s-expression
```
